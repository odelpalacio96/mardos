# Mardos

Sitio web de Mardos. Landing estático en HTML, CSS y JavaScript puro.

## Estructura

- `index.html` — página principal (hero, servicios, nosotros, contacto)
- `styles.css` — estilos
- `main.js` — pequeños scripts (año del footer)

## Desarrollo local

No requiere instalación. Abre `index.html` en el navegador, o levanta un
servidor simple:

```bash
python3 -m http.server 8000
# luego abre http://localhost:8000
```

## Despliegue

Conectado a Vercel: cada push a la rama `main` publica automáticamente.
